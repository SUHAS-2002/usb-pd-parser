6.4.6
  Alert Message
The Alert Message is provided to allow Port Partners to inform each other when there is a status change event. 
Some of the events are critical such as OCP, OVP and OTP, while others are informational such as change in a 
Battery's status from charging to neither charging nor discharging. 
The Alert Message Shall only be sent when the Source or Sink detects a status change. 
The Alert Message Shall contain exactly one Alert Data Object (ADO) and the format Shall be as shown in Figure 
6.29, "Alert Message" and Table 6.47, "Alert Data Object (ADO)".
