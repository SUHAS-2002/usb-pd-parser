  Message Header
Every Message Shall start with a Message Header as shown in:

and as defined in Table 6.1, "Message Header". The Message Header contains basic information about the Message 
and the PD Port Capabilities.
The Message Header May be used standalone as a Control Message when the Number of Data Objects field is zero 
or as the first part of a Data Message when the Number of Data Objects field is non-zero.
6.2.1.1.1
  Extended
The 1-bit Extended field Shall be set to zero to indicate a Control Message or Data Message and set to one to indicate 
an Extended Message.
