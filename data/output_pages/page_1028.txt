C.5.1
  Exit Mode Command request
The Initiator of the Exit Mode Command request sends a Message Header with the Number of Data Objects field 
set to 1 followed by a VDM Header with the Command Type (B7…6) set to zero indicating the Command is from an 
Initiator and the Command (B4…0) set to 5 to request the Responder to exit its Mode of operation. 
Table C.11, "Exit Mode Command request from Initiator Example" shows the contents of the key fields in the Message 
Header and VDM Header for an Initiator sending an Exit Mode Command request.
