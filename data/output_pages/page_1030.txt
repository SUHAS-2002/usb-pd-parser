C.6.1
  Attention Command request
The Initiator of the Attention Command request sends a Message Header with the Number of Data Objects field 
set to 1 followed by a VDM Header with the Command Type (B7…6) set to zero indicating the Command is from an 
Initiator and the Command (B4…0) set to 6 to request attention from the Responder. 
Table C.13, "Attention Command request from Initiator Example" shows the contents of the key fields in the Message 
Header and VDM Header for an Initiator sending an Attention Command request.
