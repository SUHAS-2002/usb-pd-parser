The Augmented Power Data Object (APDO) is defined to allow support for more than the four PDO types by 
extending the Power Data Object field from 2 to 4 bits when the B31…B30 are 11b. The generic APDO structure is 
shown in Table 6.8, "Augmented Power Data Object".
