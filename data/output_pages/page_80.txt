5.3
  Symbol Encoding
Except for the Preamble, all communications on the line Shall be encoded with a line code to ensure a reasonable 
level of DC-balance and a suitable number of transitions. This encoding makes receiver design less complicated and 
allows for more variations in the receiver design.
4b5b line code Shall be used. This encodes 4-bit data to 5-bit symbols for transmission and decodes 5-bit symbols 
to 4-bit data for consumption by the receiver.
The 4b5b code provides data encoding along with special symbols. Special symbols are used to signal Hard Reset, 
and delineate Packet boundaries (see Table 5.1, "4b5b Symbol Encoding").
