6.5
  Extended Message
An Extended Message Shall contain an Extended Message Header (indicated by the Extended field in the Message 
Header being set) and be followed by zero or more data bytes. Additional bytes that might be added to existing 
Messages in future Revision of this specification Shall be Ignored.
The format of the Extended Message is defined by the Message Header's Message Type field and is summarized in 
Table 6.53, "Extended Message Types". The Sent by column indicates entities which May send the given Message 
(Source, Sink or Cable Plug); entities not listed Shall Not issue the corresponding Message. The "Valid Start of 
Packet" column indicates the Messages which Shall only be issued in SOP Packets and the Messages which May be 
issued in SOP* Packets.
