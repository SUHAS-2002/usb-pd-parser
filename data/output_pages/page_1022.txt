Universal Serial Bus Power Delivery Specification, Revision 3.2, Version 1.1, 2024-10
C.2.2
  Discover SVIDs Command response
Table C.5, "Discover SVIDs Command response from Responder Example" shows the contents of the key fields in the 
Message Header and VDM Header for a Responder returning VDOs in response to a Discover SVIDs Command 
request. In this illustration, the value 3 in the Message Header Number of Data Objects field indicates that one VDO 
containing the supported SVIDs would be returned followed by a terminating VDO. 
Note:
The last VDO returned (the terminator of the transmission) contains zero value SVIDs. If a SVID value is 
zero, it is not used.
